```bash

```

```bash
git clone https://github.com/remvze/lowfi

npm install
npm run build
npm link
```

## ‚ú® Usage

Play a lofi radio:

```bash
lowfi play
```

Set volume:

```bash
lowfi play --volume 0.5
```

Randomly play a radio:

```bash
lowfi play --random
```

Play a custom YouTube URL:

```bash
lowfi play --url https://www.youtube.com/watch?v=8ZeC-8klo_o
```

Open the YouTube link in your browser:

```bash
lowfi open
```
